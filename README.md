# Exercicios - ArrayList (Java)
Projeto com 12 exercícios sobre ArrayList conforme o PDF fornecido.

Cada exercício está em um arquivo Java independente (sem package) dentro de `src/`.
Para compilar e rodar (exemplo com Ex1):
```bash
javac src/Ex1_ListaDeTarefas.java
java -cp src Ex1_ListaDeTarefas
```

Arquivos:
1. Ex1_ListaDeTarefas.java
2. Ex2_CadastroDeNomesSemDuplicatas.java
3. Ex3_ControleDeGastos.java
4. Ex4_FiltrarPalavrasCurtas.java
5. Ex5_CatalogoDeProdutos.java
6. Ex6_AgendaDeContatos.java
7. Ex7_RemoverItensPorCriterio.java
8. Ex8_BuscaDeOcorrencias.java
9. Ex9_MenuDeProdutos.java
10. Ex10_EmailsNormalizadosUnicos.java
11. Ex11_TemperaturasSemanaisAlertas.java
12. Ex12_FilaDeAtendimento.java

Observações:
- Os programas lêem entradas pelo `Scanner` (System.in). Siga as instruções exibidas ao rodar cada programa.
- Código escrito de forma clara e com comentários mínimos para facilitar estudo e upload no GitHub.
